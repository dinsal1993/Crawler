Performance

The Crawler took approximately 18 minutes for the two queries.
The time is dependent on several factors:

Number of products returned
Access time to each product
CPU Speed
Server load

We can improve the time it takes the crawler to get the results using parallel computing.
Since each page is independent of the others, if we utilize several proccessors we can cut the time by more
than 50%, assuming Asos does not block us from making HTTP requests.
We can also use a low-level, faster programming language to access the data, and than use python only for
proccessing.
